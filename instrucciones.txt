- VERSION ANGULAR // Abrimos la terminal, ejecutamos el comando "npm install" (en caso de no funcionar el anterior ingrese el siguiente comando: " npm install -g @angular/cli@latest " ) que instala las dependencias necesarias (que no fueron incluidas en la carpeta del proyecto para reducir el peso del archivo .rar); luego ejecutamos el comando "ng serve" y desde el navegador ingresamos al "http://localhost:4200".
- VERSION REACT // Las dependecias estan incluidas en la carpeta del proyecto, solo debemos ejecutar el comando en la terminal "npm start" y desde el navegador ingresamos al "http://localhost:8080".
